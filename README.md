## Instruções GIT

| Intruções| Descrição |
|-|-|
|git init|Iniciar o monitoramento do projeto|
|git status| Verifica o estado dos arquivos do presente|
|git add| Prepara os arquivos para uma nova versão|
|git comit -m "mensagem"| Cria uma nova versão.|
